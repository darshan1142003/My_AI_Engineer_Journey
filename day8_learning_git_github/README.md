Important Git Commands (Essential for Freshers)

git init
This command creates a new Git repository in the current folder.

git status
This command shows the current state of files, such as modified, new, or staged files.

git add .
This command adds all changed files to the staging area.

git commit -m "message"
This command saves the staged changes with a short description.

git remote add origin repository_url
This command connects the local project to a remote GitHub repository.

git branch -M main
This command sets the main branch name to main.

git push -u origin main
This command uploads local commits to the GitHub repository.

git pull origin main
This command downloads and merges the latest changes from GitHub.
